# 📁 منظم الملفات الذكي - Smart File Organizer

## ✨ الميزات

- 🎨 واجهة احترافية وجميلة باللغة العربية
- 📊 عرض إحصائيات مفصلة عن الملفات المنقولة
- 🎯 تنظيم ذكي للملفات حسب الصيغة
- ⚡ سريع وفعال
- 🛡️ معالجة آمنة للأخطاء

---

## 🚀 كيفية الاستخدام

### الطريقة 1: من خلال الواجهة الويب (الأفضل)

1. **تثبيت المتطلبات:**
   ```bash
   npm install
   ```

2. **تشغيل الخادم:**
   ```bash
   npm start
   ```

3. **افتح المتصفح واذهب إلى:**
   ```
   http://localhost:3000
   ```

4. أدخل مسار المجلد الذي تريد تنظيمه:
   ```
   C:\Users\YourUsername\Downloads
   ```

5. اضغط على **"✨ تنظيم الملفات الآن"**

6. سيتم تنظيم الملفات وعرض التقرير النهائي

### الطريقة 2: من خلال Command Line

```bash
cd c:\mycode\output
smartfileorganize.exe C:\Users\YourUsername\Downloads
```

---

## 📂 تصنيفات الملفات

البرنامج ينظم الملفات إلى 4 مجلدات رئيسية:

| المجلد | الملفات |
|--------|--------|
| 📄 **Documents** | `.pdf`, `.docx` |
| 🖼️ **Images** | `.jpg`, `.png` |
| 💻 **Programming** | `.cpp`, `.py` |
| 📦 **Others** | الملفات الأخرى |

---

## 💻 المتطلبات

- Windows 10 أو أحدث

## 🎯 أمثلة استخدام

```
المسار المدخل: C:\Users\Ahmed\Downloads
النتيجة:
  - Documents/     ← الملفات PDF و DOCX
  - Images/        ← الصور JPG و PNG
```

## 📞 دعم إضافي

**مشاكل شائعة:**

Q: لا يعمل البرنامج على مجلد معين؟
A: تأكد من أن المسار صحيح تماماً (لا توجد مسافات إضافية)

Q: كيف أتراجع عن التنظيم؟
A: يمكنك نقل الملفات يدوياً للخلف أو استخدام Undo من File Explorer

Q: هل يمسح الملفات الأصلية؟
A: لا، البرنامج ينقل الملفات فقط، لا يحذفها
